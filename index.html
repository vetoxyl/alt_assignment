Welcome to Okunola Abiola Landing Page

The objective of this task is to set up a web server and deploy a simple landing page to demonstrate your team's capabilities to potential investors. To achieve this, you will provision a Linux server on a cloud platform like AWS, install a web server such as Apache or Nginx, and create a basic HTML page that includes your name, a project title (“Welcome to [Your Name] Landing Page”), a brief project description, and your full bio with interesting details. Once the HTML page is created, you will deploy it on the server, configure the server to allow HTTP traffic on port 80, and provide the public IP address or URL so the page can be accessed from any browser. Deliverables include the public IP address or URL of your web page, a screenshot showing your HTML page in a browser, and clear, step-by-step documentation in a README.md file hosted in a Git repository, detailing how you provisioned the server, installed the web server, deployed the HTML page, and configured networking. For extra credit, you can optionally configure HTTPS for your web server using a free SSL certificate like Let’s Encrypt.
